# Amathia

## Descrizione
Amathia è un'app mobile in Flutter che permette agli utenti di scoprire borghi, ricette, monumenti e paesaggi naturali del Salento. Gli utenti possono salvare i luoghi nei preferiti o creare itinerari personalizzati. L'app utilizza **Supabase** per la gestione dell'autenticazione e del database e flutter come frontend per lo sviluppo dell'applicazione mobile.

## Funzionalità
- Esplorazione di borghi, ricette, monumenti e paesaggi naturali del Salento
- Possibilità di salvare i luoghi nei preferiti
- Creazione e gestione di itinerari personalizzati
